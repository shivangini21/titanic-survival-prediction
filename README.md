# Titanic Survival Prediction

Built a binary classification model to predict passenger survival 
on the Titanic dataset using Machine Learning.

## What I did
- Loaded and explored real passenger data (891 records)
- Performed data cleaning (handled missing values, dropped high-null columns)
- Encoded categorical variables (Sex: male/female → 0/1)
- Selected relevant features for training
- Trained a Random Forest Classifier using Scikit-learn

## Result
Achieved **80.4% accuracy** on the validation set.

## Tools Used
- Python, Pandas, NumPy, Scikit-learn
- Kaggle Notebooks
